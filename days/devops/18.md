## üìö Dictionary for Developers

*   **Build Step:** A step in a CI workflow that compiles code or prepares the application for deployment.
*   **Test Step:** A step in a CI workflow that runs automated tests to verify code quality.
*   **Matrix Build:** Running the same workflow across multiple configurations (versions, operating systems) simultaneously.
*   **Build Artifact:** Files generated during the build process (compiled code, reports) that are saved for later use.
*   **CI Pipeline:** A series of automated steps (build, test, etc.) that run when code changes are committed.
