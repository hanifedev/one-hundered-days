## ğŸ“š Dictionary for Developers

*   **`terraform init`:** Command that initializes a Terraform working directory, downloading providers and setting up the backend.
*   **`terraform plan`:** Command that creates an execution plan, showing what Terraform will do before making changes.
*   **`terraform apply`:** Command that applies the Terraform configuration, creating or updating infrastructure.
*   **`terraform destroy`:** Command that removes all infrastructure defined in the Terraform configuration.
*   **Terraform Backend:** Where Terraform stores its state file (local file, S3, etc.).
*   **Execution Plan:** A preview of what Terraform will do, shown before applying changes.
