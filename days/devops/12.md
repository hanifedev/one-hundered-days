## ğŸ“š Dictionary for Developers

*   **`docker pull`:** Command to download an image from a registry (like Docker Hub).
*   **`docker images`:** Command to list all Docker images on your system.
*   **`docker run`:** Command to create and start a container from an image.
*   **Port Mapping:** Mapping a port on the host machine to a port in the container (e.g., `-p 8080:80`).
*   **Detached Mode (`-d`):** Running a container in the background, detached from the terminal.
*   **`docker ps`:** Command to list running containers.
*   **`docker stop`:** Command to stop a running container.
*   **`docker rm`:** Command to remove a container.
