## üìö Dictionary for Developers

*   **External API:** An API provided by a third-party service that your application consumes.
*   **API Client:** Code that makes requests to external APIs with proper error handling and retry logic.
*   **Webhook:** A callback mechanism where external services send HTTP requests to notify your application of events.
*   **Webhook Signature:** A cryptographic signature that verifies webhook requests are authentic.
*   **Retry Logic:** Automatically retrying failed API requests with exponential backoff.
