## üìö Dictionary for Developers

*   **GraphQL Subscription:** A real-time feature that allows clients to receive updates when data changes.
*   **N+1 Query Problem:** A performance issue where N additional queries are executed for N items.
*   **DataLoader:** A utility for batching and caching database queries to solve the N+1 problem.
*   **Query Depth:** The number of nested levels in a GraphQL query.
*   **GraphQL Directive:** A way to add metadata or modify query execution behavior.
