## üìñ Task Explanations

OAuth enables users to authenticate using third-party services (Google, Facebook, GitHub), improving user experience by eliminating the need to create new accounts. Understanding OAuth is essential for modern applications that want to offer social login options.

OAuth 2.0 is the industry standard for authorization, and OpenID Connect extends it for authentication. Understanding these protocols enables you to integrate with any OAuth provider.

## üìö Dictionary for Developers

*   **OAuth 2.0:** An authorization framework that allows third-party services to access user resources.
*   **OpenID Connect:** An authentication layer built on OAuth 2.0.
*   **Social Authentication:** Allowing users to log in using social media accounts (Google, Facebook, etc.).
*   **OAuth Provider:** A service that provides OAuth authentication (Google, Facebook, GitHub).
*   **Authorization Code Flow:** The most common OAuth flow for web applications.
