## ğŸ“š Dictionary for Developers

*   **Serverless:** A cloud computing model where code runs in stateless compute containers managed by a cloud provider.
*   **Function as a Service (FaaS):** A serverless computing service that executes functions in response to events.
*   **Cold Start:** The delay when a serverless function is invoked after being idle.
*   **Stateless:** Not maintaining state between requests, required for serverless functions.
*   **Auto-Scaling:** Automatically adjusting resources based on demand.
