## üìö Dictionary for Developers

*   **OWASP Top 10:** The ten most critical web application security risks identified by OWASP.
*   **Security Header:** HTTP headers that enhance security (CSP, HSTS, X-Frame-Options, etc.).
*   **Security Audit:** A systematic review of an application to identify security vulnerabilities.
*   **Vulnerability:** A weakness in an application that could be exploited by attackers.
