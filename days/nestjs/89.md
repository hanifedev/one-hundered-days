## üìö Dictionary for Developers

*   **Memory Leak:** A bug where memory is allocated but never freed, causing memory usage to grow over time.
*   **Memory Profiling:** Analyzing memory usage to identify leaks and excessive consumption.
*   **Garbage Collection:** Automatic memory management that frees memory no longer in use.
*   **Resource Optimization:** Improving how applications use CPU, memory, and I/O resources.
*   **Heap Memory:** Memory used for dynamic allocation in Node.js applications.
