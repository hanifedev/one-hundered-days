## üìö Dictionary for Developers

*   **Offline-First:** An application design approach where the app is designed to work primarily with local data, syncing with servers when possible.
*   **Data Persistence:** The ability of data to remain stored and accessible after the application is closed and reopened.
*   **State Synchronization:** The process of keeping UI state in sync with the underlying data source (database, API, etc.).
*   **Clean Architecture:** A software design approach that separates concerns into distinct layers (UI, business logic, data) for better maintainability.
*   **Production Application:** An application that is ready for real-world use, with proper architecture, error handling, and data persistence.
