## üìñ Task Explanations

Today, you'll learn about `CustomPaint`, which allows you to draw custom graphics directly on the screen. While Flutter provides many built-in widgets, sometimes you need custom graphics that aren't available as widgets.

`CustomPaint` uses a `CustomPainter` class that defines what to draw. The `CustomPainter` receives a `Canvas` object, which provides methods for drawing:
- **`drawLine()`:** Draw lines
- **`drawCircle()`:** Draw circles
- **`drawRect()`:** Draw rectangles
- **`drawPath()`:** Draw custom paths (complex shapes)
- **`drawText()`:** Draw text
- And many more...

You'll create a class that extends `CustomPainter` and implements the `paint()` method, where you use the `Canvas` to draw your graphics. You can also implement `shouldRepaint()` to optimize when the custom paint needs to be redrawn.

This is useful for creating custom charts, progress indicators, decorative elements, or any graphics that aren't available as standard widgets.

## üìö Dictionary for Developers

*   **`CustomPaint`:** A widget that allows you to draw custom graphics using the Canvas API.
*   **`CustomPainter`:** A class that defines custom drawing logic, used with `CustomPaint` to create custom graphics.
*   **Canvas API:** A set of methods for drawing graphics (lines, shapes, text) on a canvas.
*   **`paint()` Method:** The method in `CustomPainter` where you define what to draw using the Canvas.
*   **Custom Graphics:** Visual elements drawn programmatically rather than using pre-built widgets.
