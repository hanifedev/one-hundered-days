## ğŸ“š Dictionary for Developers

*   **`setState()`:** A method in `StatefulWidget` that notifies Flutter that the widget's state has changed and triggers a rebuild of the widget tree.
*   **State Lifting:** The pattern of moving state from a child widget to a parent widget so it can be shared with multiple children.
*   **Local State:** State that is managed within a single widget and doesn't need to be shared with other widgets.
*   **Widget Lifecycle:** The sequence of methods called during a widget's lifetime (e.g., `initState`, `build`, `dispose`).
