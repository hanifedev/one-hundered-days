## üìñ Task Explanations

Expo provides access to many device features through its SDK. Haptics provide tactile feedback that enhances user experience (like vibration when pressing a button). File system access allows you to save and read files, which is useful for caching, offline functionality, and data persistence.

Understanding what device features are available through Expo helps you build more capable apps. While not every app needs every feature, knowing what's available helps you make informed decisions about app functionality.

## üí° What You'll Gain

## üìö Dictionary for Developers

*   **Haptics:** Tactile feedback (vibrations) that provide physical sensations to users.
*   **File System:** The device's storage system where files can be read and written.
*   **Device Information:** Data about the device (model, OS version, manufacturer, etc.).
*   **Expo Module:** A package in the Expo SDK that provides access to a specific device feature or capability.
*   **Native Capability:** A device feature that requires native code to access.
