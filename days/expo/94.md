## üìñ Task Explanations

Debugging performance issues requires the right tools. React DevTools helps identify unnecessary re-renders. Flipper provides comprehensive debugging capabilities including network monitoring, layout inspection, and performance profiling. Understanding how to use these tools is essential for optimizing apps.

Performance monitoring helps you measure improvements and identify regressions. Profiling tools show you where time is spent, making it easier to optimize the right parts of your code. These skills are crucial for maintaining app performance as features are added.

## üìö Dictionary for Developers

*   **Performance Profiling:** The process of measuring and analyzing app performance to identify bottlenecks.
*   **React DevTools:** Browser extension that helps debug React applications, including performance analysis.
*   **Flipper:** A platform for debugging mobile apps, providing tools for network, layout, and performance inspection.
*   **Frame Rate:** The number of frames rendered per second (60fps is the target for smooth animations).
*   **Performance Bottleneck:** A part of the code that significantly slows down app performance.
