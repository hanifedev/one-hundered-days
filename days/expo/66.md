## ğŸ“š Dictionary for Developers

*   **Unit Test:** A test that verifies a single unit of code (like a function) works correctly in isolation.
*   **Jest:** A JavaScript testing framework developed by Facebook, commonly used for React and React Native applications.
*   **Test Case:** A single test that verifies a specific behavior or functionality.
*   **Assertion:** A statement that checks if a condition is true (e.g., `expect(value).toBe(5)`).
*   **Test Suite:** A collection of related test cases grouped together.
