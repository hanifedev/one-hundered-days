## üìö Dictionary for Developers

*   **Loading State:** A state variable that tracks whether an asynchronous operation (like an API request) is currently in progress.
*   **Loading Indicator:** A visual element (like a spinner or progress bar) that shows users that an operation is in progress.
*   **Skeleton Screen:** A placeholder UI that shows the structure of content while it's loading, providing better UX than a blank screen.
*   **Error State:** A state variable that tracks whether an operation failed and stores error information.
*   **Error Handling:** The process of catching, processing, and responding to errors in a way that prevents crashes and informs users.
