## üìö Dictionary for Developers

*   **Image Optimization:** Techniques for reducing image file size while maintaining quality.
*   **Lazy Loading:** Loading images only when they're about to be visible, improving initial load time.
*   **Image Caching:** Storing loaded images in memory or disk to avoid reloading them.
*   **Bundle Size:** The total size of an app's code and assets, affecting download and install time.
*   **Memory Management:** Controlling how much memory an app uses to prevent crashes and slowdowns.
