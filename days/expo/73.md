## ğŸ“š Dictionary for Developers

*   **Test Wrapper:** A component or utility that provides dependencies (context, store, navigation) to components in tests.
*   **Mock Context:** A fake context value provided to components during testing.
*   **Mock Store:** A fake Redux store used for testing components connected to Redux.
