## üìñ Task Explanations

Moti is a declarative animation library built on top of Reanimated that simplifies creating animations. Instead of managing shared values and animated styles, Moti provides a simpler, more intuitive API. It's great for developers who want powerful animations with less boilerplate.

While Reanimated gives you more control and power, Moti provides a better developer experience for common animation patterns. Understanding both tools helps you choose the right one for each project.

## üìö Dictionary for Developers

*   **Moti:** A declarative animation library for React Native built on Reanimated.
*   **Declarative API:** An API where you describe what you want (the end state) rather than how to achieve it (imperative).
*   **Animation Props:** Properties you pass to components to define animations (like `from`, `to`, `animate`).
*   **Boilerplate:** Repetitive code that must be written to accomplish common tasks.
*   **Developer Experience:** How easy and pleasant it is to use a tool or library.
