## üìö Dictionary for Developers

*   **Animation:** The illusion of motion created by rapidly changing visual properties over time.
*   **Animated API:** React Native's built-in API for creating animations.
*   **Animated Value:** A value that can be animated over time (e.g., opacity, position, scale).
*   **`Animated.timing()`:** A function that animates a value over a specified duration using an easing function.
*   **Easing Function:** A function that determines how an animation accelerates and decelerates.
