## üìñ Task Explanations

Custom hooks are a powerful React feature that allows you to extract component logic into reusable functions. They let you share stateful logic between components without duplicating code. Custom hooks must follow the rules of hooks (start with "use", only call other hooks from within them).

Creating custom hooks for common patterns (like AsyncStorage, API calls, form handling) makes your code more maintainable and testable. They're one of the best ways to share logic in React applications and are widely used in production codebases.

## üìö Dictionary for Developers

*   **Custom Hook:** A JavaScript function that starts with "use" and may call other hooks, allowing you to extract and reuse component logic.
*   **Stateful Logic:** Logic that involves state management, side effects, or other React hooks.
*   **Hook Rules:** The rules that must be followed when using hooks (only call at top level, only call from React functions).
*   **Logic Extraction:** The process of moving logic out of components into reusable functions (like custom hooks).
*   **Reusable Code:** Code that can be used in multiple places without duplication, improving maintainability.
