## üìñ Task Explanations

Reanimated provides powerful animation functions like `withSpring` and `withTiming` that work with shared values. These functions run on the UI thread, providing smooth animations. Interpolation in Reanimated is similar to Animated API but more powerful and performant.

## üìö Dictionary for Developers

*   **`withSpring`:** A Reanimated function that creates spring-based animations.
*   **`withTiming`:** A Reanimated function that creates timing-based animations.
*   **`withSequence`:** A Reanimated function that chains animations together.
*   **`withDelay`:** A Reanimated function that delays the start of an animation.
*   **Extrapolate:** A configuration that determines how values outside the interpolation range are handled.
