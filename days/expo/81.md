## üìñ Task Explanations

React Native Reanimated is a more powerful animation library that runs animations on the UI thread, providing better performance and smoother animations. Unlike Animated API, Reanimated allows you to write animations that run entirely on the native side, keeping the JavaScript thread free.

Worklets are functions that can run on the UI thread, enabling complex animations without blocking JavaScript. Shared values are the Reanimated equivalent of Animated values, but they're more powerful and can be accessed from both JavaScript and the UI thread.

## üìö Dictionary for Developers

*   **React Native Reanimated:** A high-performance animation library that runs animations on the UI thread.
*   **Worklet:** A function that can run on the UI thread, enabling animations without blocking JavaScript.
*   **Shared Value:** A Reanimated value that can be accessed from both JavaScript and the UI thread.
*   **`useSharedValue`:** A hook that creates a shared value for animations.
*   **`useAnimatedStyle`:** A hook that creates animated styles based on shared values.
*   **UI Thread:** The native thread where UI updates happen, separate from the JavaScript thread.
