## üìö Dictionary for Developers

*   **Interpolation:** The process of mapping one range of values to another range (e.g., mapping 0-1 to 0-360 for rotation).
*   **`interpolate()`:** A function that creates a mapping between input and output ranges for animations.
*   **Transform:** CSS-like properties that modify an element's appearance (translate, scale, rotate).
*   **Transform Animation:** Animating transform properties to create motion, scaling, or rotation effects.
*   **Coordinate Animation:** Animating multiple properties together to create coordinated, complex animations.
