## üìñ Task Explanations

Connecting Redux to React Native components is done through the `react-redux` library. The `Provider` component makes the Redux store available to all components in your app. `useSelector` lets components read data from the store, and `useDispatch` lets them dispatch actions to update the store.

Selectors are functions that extract specific pieces of data from the store. They can also compute derived data (like filtering a list or calculating totals). Using selectors keeps your components clean and makes your data access logic reusable and testable.

## üìö Dictionary for Developers

*   **Redux Provider:** A component from `react-redux` that makes the Redux store available to all child components.
*   **`useSelector` Hook:** A React hook from `react-redux` that allows components to read data from the Redux store.
*   **`useDispatch` Hook:** A React hook from `react-redux` that returns a reference to the `dispatch` function for dispatching actions.
*   **Selector:** A function that extracts specific data from the Redux store, often with some transformation or computation.
*   **Derived State:** Data computed from other state (e.g., total price calculated from cart items).
*   **Store Subscription:** The mechanism by which components automatically re-render when relevant parts of the store change.
