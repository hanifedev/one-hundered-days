## üìñ Task Explanations

Different animation types create different feelings: timing animations are predictable and controlled, spring animations feel natural and bouncy, and decay animations simulate momentum. Understanding when to use each type helps you create animations that feel right for your use case.

Easing functions control how animations accelerate and decelerate. The right easing function can make an animation feel natural and polished. Linear easing feels robotic, while ease-in-out feels smooth and natural for most UI animations.

## üìö Dictionary for Developers

*   **Timing Animation:** An animation that changes a value over a fixed duration with an easing function.
*   **Spring Animation:** A physics-based animation that simulates a spring, creating natural, bouncy motion.
*   **Decay Animation:** An animation that simulates momentum, gradually slowing down (like a scroll that comes to rest).
*   **Easing Function:** A function that determines the rate of change of an animation over time.
*   **Animation Duration:** The length of time an animation takes to complete.
