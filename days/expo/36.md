## ğŸ“– Task Explanations

Navigation is fundamental to mobile app development. Unlike web apps where URLs handle navigation, mobile apps use a navigation stack managed by libraries like React Navigation. Today you'll set up React Navigation, which is the industry standard for React Native navigation.

The `NavigationContainer` is the root component that manages the navigation state for your entire app. Understanding this setup is crucial before you can create navigators and navigate between screens. This foundation will enable you to build complex navigation structures in the coming days.
