## üìö Dictionary for Developers

*   **`TextInput`:** A React Native component that allows users to enter text. It's the primary component for collecting text-based user input.
*   **Controlled Component:** A component whose value is controlled by React state, giving you full control over its behavior.
*   **`keyboardType`:** A prop that determines which keyboard type appears on mobile devices (e.g., `email-address`, `numeric`, `phone-pad`).
*   **`secureTextEntry`:** A prop that hides the text input (useful for passwords).
*   **`onChangeText`:** A callback function that is called whenever the text in the input changes.
