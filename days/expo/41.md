## üìö Dictionary for Developers

*   **HTTP (Hypertext Transfer Protocol):** The protocol used for communication between clients (like your app) and servers.
*   **API (Application Programming Interface):** A set of rules and protocols that allows different software applications to communicate with each other.
*   **`fetch()`:** A built-in JavaScript function for making HTTP requests.
*   **GET Request:** An HTTP method used to retrieve data from a server.
*   **JSON (JavaScript Object Notation):** A lightweight data format commonly used for API responses.
*   **Status Code:** A numeric code that indicates the result of an HTTP request (e.g., 200 = success, 404 = not found, 500 = server error).
