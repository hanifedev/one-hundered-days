## ğŸ“š Dictionary for Developers

*   **OTA Update (Over-the-Air Update):** Updating an app's JavaScript and assets without going through app store review.
*   **EAS Update:** Expo's service for publishing OTA updates to apps.
*   **Update Channel:** A named channel (like "production" or "preview") that determines which updates users receive.
*   **App Store Review:** The process where app stores review updates before making them available (not required for OTA updates).
