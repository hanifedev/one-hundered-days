## ğŸ“– Task Explanations

SQLite is a lightweight, embedded database that's perfect for mobile apps when you need to store structured data with relationships and complex queries. While AsyncStorage is great for simple key-value storage, SQLite is better for apps that need to query, filter, and relate data (like a todo app with categories, a notes app with tags, or an expense tracker).

Expo provides `expo-sqlite` which makes it easy to use SQLite in your apps. Understanding SQL basics (SELECT, INSERT, UPDATE, DELETE) is essential for working with SQLite. This knowledge is valuable for building data-intensive mobile applications.

## ğŸ“š Dictionary for Developers

*   **SQLite:** A lightweight, embedded relational database management system that stores data in a single file.
*   **Database Schema:** The structure of a database, including tables, columns, data types, and relationships.
*   **SQL (Structured Query Language):** A language used to communicate with databases for querying and manipulating data.
*   **Table:** A collection of related data organized in rows and columns in a database.
*   **Query:** A request for data or information from a database.
*   **Embedded Database:** A database that runs in the same process as the application, without requiring a separate server.
