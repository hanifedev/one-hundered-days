## üìñ Task Explanations

Push notifications are essential for engaging users and keeping them informed. Expo Notifications makes it relatively easy to implement push notifications without writing native code. You can send notifications from your server or schedule local notifications that don't require internet.

Understanding the notification flow (registering device, getting tokens, sending notifications) is important for production apps. Push notifications are a key feature for many app types (messaging, social media, productivity, e-commerce).

## üìö Dictionary for Developers

*   **Push Notification:** A message sent from a server to a user's device, even when the app isn't open.
*   **Local Notification:** A notification scheduled and displayed by the app itself, without server involvement.
*   **Device Token:** A unique identifier for a device that allows servers to send notifications to it.
*   **Notification Permission:** User authorization required to display notifications.
*   **Expo Push Notification Service:** Expo's service for sending push notifications to devices.
