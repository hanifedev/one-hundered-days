## üìö Dictionary for Developers

*   **Expo Location:** An Expo package that provides access to device GPS and location services.
*   **GPS Coordinates:** Latitude and longitude values that specify a location on Earth.
*   **Geocoding:** Converting addresses to coordinates (or vice versa).
*   **Location Permission:** User authorization required to access device location.
*   **Map Component:** A component that displays an interactive map with markers, routes, and other features.
