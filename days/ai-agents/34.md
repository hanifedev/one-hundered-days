## ğŸ’¡ Today's Dictionary

*   **Event-Driven Architecture:** An architectural pattern where agents respond to events (occurrences, changes) rather than polling or scheduled execution.
*   **Event Source:** A system or component that generates events that agents can react to (user actions, APIs, sensors, etc.).
*   **Reactive Agent:** An agent that responds to events in real-time, triggering actions based on event types and content.
*   **Event Processing:** The handling of events by agents, including filtering, transformation, routing, and action triggering.
