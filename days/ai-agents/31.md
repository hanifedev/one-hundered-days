## ğŸ’¡ Today's Dictionary

*   **Hierarchical Agent:** An agent architecture where agents are organized in layers, with higher-level agents (supervisors) managing lower-level agents (workers).
*   **Supervisor Agent:** An agent that receives tasks, decomposes them, and delegates subtasks to specialized worker agents.
*   **Worker Agent:** A specialized agent that performs specific tasks delegated by a supervisor agent.
*   **Task Decomposition:** The process of breaking down a complex task into smaller, manageable subtasks.
