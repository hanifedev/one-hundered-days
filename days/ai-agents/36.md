## üí° Today's Dictionary

*   **Containerization:** Packaging an application and its dependencies into a container (Docker) for consistent deployment across environments.
*   **Container Orchestration:** Managing multiple containers (starting, stopping, scaling, networking) using tools like Kubernetes or Docker Compose.
*   **Dockerfile:** A file that defines how to build a Docker container image for an application.
*   **Production Deployment:** Deploying an application to a production environment where it serves real users.
