## üí° Today's Dictionary

*   **Protocol:** A set of rules and conventions that define how agents communicate and exchange information.
*   **Message Queue:** A communication mechanism that allows agents to send and receive messages asynchronously.
*   **Publish-Subscribe (Pub/Sub):** A messaging pattern where agents can publish messages to topics and subscribe to receive messages from topics of interest.
