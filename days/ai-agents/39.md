## üí° Today's Dictionary

*   **Horizontal Scaling:** Adding more instances of an application to handle increased load (scaling out).
*   **Vertical Scaling:** Increasing the resources (CPU, memory) of a single instance to handle increased load (scaling up).
*   **Performance Optimization:** Improving the speed, efficiency, and resource usage of agent systems.
*   **Load Testing:** Testing a system under expected or higher loads to identify performance bottlenecks and capacity limits.
*   **Cost Optimization:** Strategies for reducing the cost of running agent systems, particularly LLM API costs.
