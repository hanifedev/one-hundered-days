## üí° Today's Dictionary

*   **Code-Aware Agent:** An AI agent specifically designed to understand, analyze, and interact with source code.
*   **File System Tools:** Tools that allow an agent to interact with the local file system (e.g., `read_file`, `list_directory`).
*   **Shell Tool:** A tool that allows an agent to execute shell commands, which is extremely powerful but also very dangerous.
