## üí° Today's Dictionary

*   **Agent Evaluation:** The systematic assessment of agent performance using metrics, tests, and benchmarks.
*   **Benchmarking:** Comparing agent performance against standard tasks or other agents to measure relative performance.
*   **A/B Testing:** Comparing two versions of an agent (different models, prompts, or architectures) to determine which performs better.
*   **Evaluation Framework:** A structured approach and set of tools for evaluating agent performance across different dimensions.
