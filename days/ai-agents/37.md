## üí° Today's Dictionary

*   **Agent API:** An application programming interface that exposes agent functionality for use by other systems and applications.
*   **REST API:** A web API that follows REST (Representational State Transfer) principles, using HTTP methods (GET, POST, etc.).
*   **Async API:** An API that handles long-running operations asynchronously, often using background jobs and webhooks for completion.
*   **API Documentation:** Documentation that describes how to use an API, including endpoints, parameters, and response formats (often using OpenAPI/Swagger).
