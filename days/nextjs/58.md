## üìö Dictionary for Developers

*   **Headless CMS:** A content management system that provides content via APIs without a built-in frontend.
*   **Content Modeling:** The process of structuring content types and fields in a CMS.
*   **Content API:** An API provided by a CMS for fetching and managing content.
*   **Content Preview:** A feature that allows content editors to preview content before publishing.
*   **Content-Driven Application:** An application where content is managed separately from code, typically using a CMS.
