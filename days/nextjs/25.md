## ğŸ“š Dictionary for Developers

*   **Dynamic Route:** A route that matches multiple URLs using parameters (e.g., `/posts/[id]`).
*   **Route Group:** A way to organize routes in folders without affecting the URL structure using `(folder)` syntax.
*   **Parallel Route:** A route that renders simultaneously with other routes using `@folder` syntax.
*   **Intercepting Route:** A route that intercepts navigation to show different UI (like modals) using `(.)folder` syntax.
*   **Route Parameter:** A dynamic segment in a URL that can be accessed in the component (e.g., `params.id`).
