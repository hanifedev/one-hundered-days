## ğŸ“š Dictionary for Developers

*   **API Route:** An endpoint in your Next.js application that handles HTTP requests and returns responses.
*   **Server Action:** A server function that can be called directly from Client Components, simplifying form handling and data mutations.
*   **Progressive Enhancement:** Building applications that work without JavaScript but are enhanced when JavaScript is available.
*   **Data Mutation:** The process of creating, updating, or deleting data (as opposed to reading data).
*   **Form Action:** A Server Action that handles form submissions directly, without needing a separate API route.
