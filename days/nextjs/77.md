## üìö Dictionary for Developers

*   **Progressive Web App (PWA):** A web application that uses modern web capabilities to provide an app-like experience.
*   **Service Worker:** A script that runs in the background, enabling offline functionality and advanced caching.
*   **Web App Manifest:** A JSON file that provides metadata about a web app, making it installable.
*   **Offline Support:** The ability of an application to work without an internet connection.
*   **Installable Web App:** A web app that can be installed on a device like a native app.
