## üìö Dictionary for Developers

*   **Environment Variable:** A variable whose value is set outside the application, typically for configuration.
*   **Security Header:** HTTP headers that provide security features (CSP, X-Frame-Options, etc.).
*   **Content Security Policy (CSP):** A security feature that helps prevent XSS attacks by restricting resource loading.
*   **CORS (Cross-Origin Resource Sharing):** A mechanism that allows resources to be requested from different origins.
*   **CSRF (Cross-Site Request Forgery):** An attack that tricks users into performing actions they didn't intend.
