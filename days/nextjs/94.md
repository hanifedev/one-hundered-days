## üìö Dictionary for Developers

*   **Scaling:** The ability of an application to handle increased load (users, data, complexity).
*   **Vertical Scaling:** Adding more resources (CPU, RAM) to existing servers.
*   **Horizontal Scaling:** Adding more servers to handle increased load.
*   **Load Balancing:** Distributing traffic across multiple servers to handle high load.
*   **Load Testing:** Testing an application under expected or higher load to identify bottlenecks.
