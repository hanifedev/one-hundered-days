## üìö Dictionary for Developers

*   **Serverless:** A cloud computing model where you run code without managing servers, with automatic scaling.
*   **Serverless Function:** A function that runs in a serverless environment, scaling automatically based on demand.
*   **Edge Function:** A serverless function that runs at edge locations worldwide for low latency.
*   **Function as a Service (FaaS):** A cloud service that executes functions in response to events.
*   **Pay-Per-Use:** A pricing model where you only pay for the compute resources you actually use.
